# Week 1 Foundation Assignment

## Project
Bloch Equation Simulation and Relaxation Times using Python

## Description
This project simulates the Bloch equations for two different tissue moieties (A and B) using Python. The simulation visualizes:

- 3D magnetization vector trajectory
- Longitudinal relaxation (Mz)
- Transverse relaxation (Mxy)
- Effect of relaxation parameters (T1 and T2)
- Effect of static magnetic field (B0)
